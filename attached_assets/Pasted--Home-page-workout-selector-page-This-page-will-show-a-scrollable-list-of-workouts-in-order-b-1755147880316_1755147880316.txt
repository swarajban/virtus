### Home page / workout selector page
- This page will show a scrollable list of workouts in order by `workout_number`
- Show program name (don't need a selector as there will only be one program for now)
- There should be a visual indicator for which workouts have been completed, vs which need to be done
	- There should also be a "progress" meter to indicate how many workouts out of total workouts have been completed
- There should be a visual indicator for which workout to do next (the very next workout number that hasn't been completed yet)
- Each list item should show
	- Workout name, workout week, and workout day from JSON
	- For completed workouts, show the date workout was completed
- Clicking the workout item should take to you to the Workout page for selected workout
- There should be a link to 1RM page to view / edit 1RM weights

### Workout & exercise page
- This page will be used to view + track a workout
- Workout status: not started, in progress, or completed
	- Track timestamps for when workout was started, and when it was completed
- If the workout has not yet been started:
	- The page should show a list of all the exercises required for workout
	- There should be a button at the top that says "start workout" that logs start time and marks workout as in progress. There should also be a button that says "complete workout" that marks workout as completed (even if all exercises have not been completed)
	- Each list item should show
		- Exercise name
		- Type of set (working or warm-up)
		- Sets x reps (for example, if 5 sets with 10 reps, this should show: "5 x 10")
			- If AMRAP, show AMRAP and no set info
- When a workout is in progress, take user to next (first) exercise in workout
- Exercise page for in-progress workout
	- Show exercise name, type of set, number of sets and reps (if AMRAP, show that it's AMRAP and don't show sets/reps)
	- Sets and reps should be inputs that can be edited (but are prepopulated with workout data)
		- It should be possible to edit sets & reps by clicking a + or - button or click in using a number input form
	- Weight should be an input (in lbs) and should use a number input form
		- It should possible to adjust weight with + or - button in increments of 5 lbs
	- Show load percentage and RPE
	- If exercise name matches one of the 1RM exercises, e.g., "Back squat", automatically calculate & prepopulate weight by doing 1RM weight x load percentage rounded to nearest 5 (round up)
		- If user edits weight, show how the actual weight percentage differs from the recommended load percentage
	- Show notes in a call-out type box
	- If exercise is part of a superset (superset_label is not null), show the superset label, and an indicator that this is part of a super set
	- There should be an input for user's free text notes that can be stored / associated for this exercise on this workout (e.g., "lower weight today to focus on form")
	- There should be the following buttons / navs on the page
		- Workout: takes you back to workout page
		- Previous/Next: navigates to previous/next exercise pages (especially useful for supersets)
		- History: this should open a pop-up that renders a list of dates when this exercise.name was performed (descending order). Each list item should show
			- Date when exercise was performed
			- Sets, reps, weight, any free text notes submitted as part of past exercises
			- It should also render a line graph where x-axis is dates and y-axis is weight (axis should always start at 0lb)
		- Complete exercise: marks exercise as completed, logs exercise data, and takes user to next exercise. If this is the last exercise in the workout, take user back to exercise page
- For completed workouts
	- Show list of exercises just like when "not started", but also show actual sets x reps + weights logged
	- There should be a button to clear workout state (reset to not started, and wipe all exercise tracking)
	- Clicking in to any exercise should let user update any exercise data (eg sets, reps, weights)

### 1RM page
- This page should let the user view / edit their 1RM weight for the following lifts:
	- Back squat
	- Barbell bench press
	- Deadlift
	- Overhead press
